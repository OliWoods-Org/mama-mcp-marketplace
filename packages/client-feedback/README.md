# client feedback MCP

Customer feedback systems — NPS surveys, CSAT tracking, feature requests, feedback widgets, sentiment analysis, churn prediction

## Tools
| Tool | Description |
|------|-------------|\n| `create_nps_survey` | Generate an NPS survey with customizable touchpoints |\n| `analyze_feedback` | Analyze feedback responses for sentiment and themes |\n| `feature_request_board` | Organize and prioritize feature requests by impact |\n| `churn_predictor` | Score churn risk from feedback patterns and engagement |\n| `satisfaction_report` | Generate CSAT/CES report with trends and benchmarks |\n| `feedback_widget` | Configure an in-app feedback collection widget |

## Setup
```bash
npm install && npm run build
```
