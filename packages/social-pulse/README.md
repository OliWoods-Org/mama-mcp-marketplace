# social pulse MCP

Social listening — brand monitoring, competitor tracking, sentiment analysis, trend detection, crisis alerts, share of voice

## Tools
| Tool | Description |
|------|-------------|\n| `brand_monitor` | Track brand mentions across social platforms |\n| `competitor_tracker` | Monitor competitor social activity and engagement |\n| `sentiment_analyzer` | Analyze sentiment trends for a brand or topic |\n| `trend_detector` | Detect emerging trends and viral topics in a niche |\n| `crisis_alert` | Set up alerts for negative sentiment spikes or PR issues |

## Setup
```bash
npm install && npm run build
```
