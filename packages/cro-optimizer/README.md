# @mama/cro-optimizer-mcp

**Conversion Rate Optimization (CRO) MCP Server**

Turn more visitors into customers. Audit landing pages, design A/B tests, find funnel leaks, sharpen CTAs, and optimize your pricing page — all with actionable, data-backed recommendations.

## Tools

| Tool | Description |
|------|-------------|
| `landing_page_audit` | Score every page element, identify friction points, estimate revenue uplift |
| `ab_test_generator` | Design statistically rigorous A/B tests with sample sizes and duration |
| `funnel_analyzer` | Identify where visitors drop off and quantify the revenue opportunity |
| `cta_optimizer` | Generate 5 CTA variants with CTR scores and button design recommendations |
| `pricing_page_review` | Analyze pricing structure, psychology, and annual plan opportunity |

## Usage

```json
{
  "mcpServers": {
    "cro-optimizer": {
      "command": "npx",
      "args": ["-y", "@mama/cro-optimizer-mcp"]
    }
  }
}
```

## Example Prompts

- "Audit my SaaS landing page — our current conversion rate is 2.1% and we target free trial signups"
- "Generate an A/B test plan for my hero CTA — hypothesis is the button copy is too vague"
- "Analyze my checkout funnel: Homepage → Pricing → Signup → Payment → Confirmation"
- "Optimize my current CTA 'Get Started' for demo bookings on a B2B SaaS pricing page"
