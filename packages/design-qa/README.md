# design qa MCP

Design QA — screenshot comparison, visual regression testing, responsive audit, accessibility checks, pixel-perfect verification

## Tools
| Tool | Description |
|------|-------------|\n| `screenshot_compare` | Compare two screenshots and identify visual differences |\n| `responsive_audit` | Audit a page across breakpoints for layout issues |\n| `accessibility_check` | Run WCAG accessibility audit on a component or page |\n| `design_checklist` | Generate a design QA checklist for a feature or page |\n| `visual_regression` | Set up visual regression testing pipeline configuration |

## Setup
```bash
npm install && npm run build
```
