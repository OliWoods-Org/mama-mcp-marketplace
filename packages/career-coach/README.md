# career coach MCP

AI career advisor — resume review, interview prep, salary negotiation, career pathing, cover letters, LinkedIn optimization

## Tools
| Tool | Description |
|------|-------------|\n| `review_resume` | Score and provide feedback on a resume for a target role |\n| `interview_prep` | Generate interview questions with STAR-format answer frameworks |\n| `salary_negotiation` | Analyze an offer against market rates with negotiation script |\n| `career_path` | Map career progression from current to target role |\n| `cover_letter` | Generate a tailored cover letter from resume and job description |\n| `linkedin_optimizer` | Optimize LinkedIn headline, summary, and skills for target roles |

## Setup
```bash
npm install && npm run build
```
