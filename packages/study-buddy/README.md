# study buddy MCP

AI study assistant — concept explanations, flashcards, practice tests, study plans, note summaries

## Tools
| Tool | Description |
|------|-------------|\n| `explain_concept` | Explain a concept at the appropriate education level with analogies |\n| `generate_flashcards` | Generate question/answer flashcard pairs from a topic |\n| `practice_test` | Generate a practice test with answer key |\n| `study_plan` | Create a day-by-day study schedule for an exam |\n| `summarize_notes` | Summarize notes into key terms, concepts, and potential exam questions |

## Setup
```bash
npm install && npm run build
```
