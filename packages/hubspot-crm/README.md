# hubspot crm MCP

HubSpot CRM — contact management, deal pipeline, lead scoring, activity logging, email sequences, sales analytics

## Tools
| Tool | Description |
|------|-------------|\n| `manage_contacts` | Create, update, search, and merge contacts |\n| `deal_pipeline` | View and manage deals across pipeline stages |\n| `lead_scorer` | Score leads based on engagement signals and firmographics |\n| `log_activity` | Log calls, emails, meetings, and notes to contact timeline |\n| `email_sequence` | Create automated email sequences with personalization |\n| `sales_dashboard` | Generate sales analytics with pipeline velocity and forecasts |

## Setup
```bash
npm install && npm run build
```
