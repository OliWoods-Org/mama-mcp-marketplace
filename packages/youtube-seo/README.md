# @mama/youtube-seo-mcp

**YouTube SEO MCP Server**

Optimize every element of your YouTube presence — titles, descriptions, tags, thumbnails, and full channel audits — to maximize reach and subscriber growth.

## Tools

| Tool | Description |
|------|-------------|
| `title_optimizer` | Generate 5 CTR-scored title variants using proven hook formulas |
| `description_writer` | Write full SEO descriptions with timestamps, CTAs, and hashtags |
| `tag_generator` | Generate categorized tags with copy-paste string (under 500 chars) |
| `thumbnail_analyzer` | Score thumbnail concepts and get specific design improvements |
| `channel_audit` | Audit channel health across key metrics with a 30-day action plan |

## Usage

```json
{
  "mcpServers": {
    "youtube-seo": {
      "command": "npx",
      "args": ["-y", "@mama/youtube-seo-mcp"]
    }
  }
}
```
