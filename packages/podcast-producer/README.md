# Podcast Producer MCP (@mama/podcast-producer-mcp)

Full-stack podcast production toolkit for Claude. Plan episodes, research guests, write show notes, find viral clips, and build a growth engine — without leaving your AI assistant.

## Tools (5)

| Tool | Description |
|------|-------------|
| `episode_planner` | Plan a complete episode — hooks, segment structure with timing, interview questions, SEO metadata |
| `guest_research` | Research a guest — fit scores, expertise areas, interview angles, outreach email template |
| `show_notes` | Generate SEO-optimized show notes — summary, takeaways, timestamps, resources, and CTA |
| `clip_finder` | Identify best clips for social — viral score, platform specs, caption hooks, production checklist |
| `podcast_growth` | Build a growth strategy — benchmarks, untapped channels, quick wins, monetization readiness |
