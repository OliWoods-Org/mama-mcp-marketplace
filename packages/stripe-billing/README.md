# stripe billing MCP

Stripe SaaS billing — subscription setup, pricing tiers, usage metering, webhook handling, checkout flows, revenue analytics

## Tools
| Tool | Description |
|------|-------------|\n| `setup_subscription` | Configure a Stripe subscription with pricing tiers and trial periods |\n| `create_checkout` | Generate a Stripe Checkout session with line items and success/cancel URLs |\n| `usage_meter` | Track and report usage-based billing metrics |\n| `webhook_handler` | Generate webhook handler code for Stripe events |\n| `revenue_dashboard` | Analyze MRR, churn rate, LTV, and revenue trends |\n| `pricing_optimizer` | Compare pricing strategies and recommend optimal tier structure |

## Setup
```bash
npm install && npm run build
```
