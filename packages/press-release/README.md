# Press Release MCP (@mama/press-release-mcp)

PR & media toolkit for Claude. Write press releases, craft journalist pitches, assemble press kits, and plan distribution — all without leaving your AI assistant.

## Tools (4)

| Tool | Description |
|------|-------------|
| `write_release` | Write a complete, publication-ready press release for any announcement type |
| `media_pitch` | Craft a targeted pitch email with subject lines, outlet recommendations, and journalist tips |
| `press_kit` | Build a press kit with company overview, stats, bios, boilerplate, and media asset checklist |
| `distribution_plan` | Create a full distribution plan — timeline, wire services, outreach targets, and success metrics |

## Setup

```bash
npm install
npm run build
```

```json
{
  "mcpServers": {
    "mama-press-release": {
      "command": "node",
      "args": ["/path/to/press-release/dist/index.js"]
    }
  }
}
```
