# x growth MCP

Twitter/X growth engine — viral hooks, algorithm-optimized posts, engagement strategy, thread crafting, analytics, audience growth

## Tools
| Tool | Description |
|------|-------------|\n| `viral_post` | Generate an algorithm-optimized X post for maximum reach |\n| `thread_crafter` | Build a high-engagement thread with hooks and CTAs |\n| `engagement_strategy` | Analyze an account and recommend growth tactics |\n| `reply_optimizer` | Draft replies to trending posts for visibility |\n| `analytics_review` | Analyze post performance and recommend improvements |\n| `audience_builder` | Identify target audience segments and content themes |

## Setup
```bash
npm install && npm run build
```
