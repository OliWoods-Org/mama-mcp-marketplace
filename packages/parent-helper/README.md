# parent helper MCP

AI parenting assistant — activities, milestones, routines, behavior guidance, kid-friendly meals

## Tools
| Tool | Description |
|------|-------------|\n| `activity_finder` | Find age-appropriate activities based on weather, time, and interests |\n| `milestone_checker` | Check developmental milestones for a child's age |\n| `routine_builder` | Build a daily routine based on child age and family schedule |\n| `behavior_guide` | Get strategies for common behavior challenges by age |\n| `meal_ideas_kids` | Find kid-approved meal ideas with hidden veggie tricks |

## Setup
```bash
npm install && npm run build
```
