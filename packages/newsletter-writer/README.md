# newsletter writer MCP

AI newsletter creation — full drafts, subject lines, content calendars, audience segmentation, growth tactics

## Tools
| Tool | Description |
|------|-------------|\n| `write_newsletter` | Write a complete newsletter draft from topic and audience |\n| `subject_line_generator` | Generate 10 subject lines with estimated open rates |\n| `content_calendar` | Create a 4-week newsletter content calendar |\n| `audience_segmenter` | Segment subscribers by engagement and recommend targeting |\n| `growth_tactics` | Generate a growth action plan based on current subscriber count |

## Setup
```bash
npm install && npm run build
```
