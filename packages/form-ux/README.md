# form ux MCP

Form builder — validation patterns, multi-step wizards, conditional logic, accessibility, spam protection, submission handling

## Tools
| Tool | Description |
|------|-------------|\n| `form_builder` | Generate a form component with fields, validation, and error handling |\n| `multi_step_wizard` | Create a multi-step form wizard with progress tracking |\n| `validation_rules` | Generate Zod or Yup validation schema from field descriptions |\n| `spam_protection` | Configure honeypot, reCAPTCHA, and rate limiting for forms |\n| `submission_handler` | Generate a form submission endpoint with email notification |

## Setup
```bash
npm install && npm run build
```
