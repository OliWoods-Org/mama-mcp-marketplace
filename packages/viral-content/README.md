# viral content MCP

Viral content creation — hooks, scripts, threads, hashtags, repurposing, trend analysis for TikTok, Instagram, Twitter, LinkedIn, YouTube

## Tools
| Tool | Description |
|------|-------------|\n| `hook_generator` | Generate 10 ranked viral hooks for a topic and platform |\n| `script_writer` | Write a full video or post script from a hook |\n| `thread_builder` | Build an optimized Twitter or LinkedIn thread |\n| `hashtag_research` | Research and tier 30 hashtags for a topic and platform |\n| `content_repurposer` | Repurpose long-form content into 10+ platform-specific pieces |\n| `trend_analyzer` | Analyze trending formats and topics for a niche and platform |

## Setup
```bash
npm install && npm run build
```
