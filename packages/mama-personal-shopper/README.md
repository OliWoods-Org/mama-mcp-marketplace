# mama personal shopper MCP

AI personal shopping — product search, price comparison, deal finding, gift ideas, style matching

## Tools
| Tool | Description |
|------|-------------|\n| `search_products` | Search for products matching a description |\n| `compare_prices` | Compare prices across retailers for a product |\n| `find_deals` | Find current deals and coupons in a category |\n| `gift_finder` | Find gift ideas based on recipient profile and budget |\n| `style_match` | Find matching items for an outfit or style |\n| `track_price` | Set up a price alert for a product |

## Setup
```bash
npm install && npm run build
```
