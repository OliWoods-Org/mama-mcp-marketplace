# marketplace scan MCP

Marketplace intelligence — competitor analysis, pricing trends, review monitoring, listing optimization, market sizing

## Tools
| Tool | Description |
|------|-------------|\n| `scan_competitors` | Analyze competitor marketplace listings and pricing |\n| `price_tracker` | Track pricing trends across marketplace categories |\n| `review_monitor` | Monitor and analyze product reviews for insights |\n| `market_sizer` | Estimate market size and opportunity for a product category |\n| `listing_auditor` | Audit a marketplace listing for SEO and conversion optimization |

## Setup
```bash
npm install && npm run build
```
