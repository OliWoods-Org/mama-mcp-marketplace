# real estate analyzer MCP

Real estate investment analysis — rental yield, comps, mortgage calculator, 1031 exchanges, flip calculator, investment scoring

## Tools
| Tool | Description |
|------|-------------|\n| `rental_yield` | Calculate gross yield, net yield, cap rate, and cash-on-cash return |\n| `comp_analysis` | Estimate property value from comparable sales data |\n| `mortgage_calculator` | Calculate monthly payment, total interest, and amortization |\n| `exchange_1031_planner` | Plan a 1031 exchange with timeline, boot calculation, and QI requirements |\n| `flip_calculator` | Calculate flip profit, ROI, and risk assessment |\n| `investment_scorecard` | Score a property 0-100 across cash flow, appreciation, demand, and risk |

## Setup
```bash
npm install && npm run build
```
