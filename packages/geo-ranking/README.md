# @mama/geo-ranking-mcp

**GEO (Generative Engine Optimization) & AI Search Visibility MCP Server**

Dominate AI-generated search results. Audit your brand's presence across ChatGPT, Perplexity, Google AI Overviews, and more — then build a strategy to own the answers.

## Tools

| Tool | Description |
|------|-------------|
| `audit_ai_visibility` | Audit AI visibility score by engine, citation rate, strengths & gaps |
| `optimize_content` | Optimize content for AI citation with structure, schema & keyword guidance |
| `competitor_ai_presence` | Map competitors' AI search presence and citation tactics |
| `citation_opportunities` | Find prioritized citation-building opportunities by effort & impact |
| `ai_search_strategy` | Build a phased AI search domination strategy with KPIs and action plan |

## Usage

```json
{
  "mcpServers": {
    "geo-ranking": {
      "command": "npx",
      "args": ["-y", "@mama/geo-ranking-mcp"]
    }
  }
}
```
