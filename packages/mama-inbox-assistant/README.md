# mama inbox assistant MCP

AI inbox management — email triage, draft replies, action item extraction, follow-up reminders

## Tools
| Tool | Description |
|------|-------------|\n| `triage_emails` | Prioritize and categorize a list of emails by urgency and type |\n| `draft_reply` | Generate a context-aware reply to an email |\n| `extract_action_items` | Extract action items and deadlines from an email thread |\n| `follow_up_reminder` | Generate a follow-up draft for a sent email |\n| `meeting_from_email` | Extract meeting details from an email thread |

## Setup
```bash
npm install && npm run build
```
