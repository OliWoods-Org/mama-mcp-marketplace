# data viz MCP

Charts and dashboards — Recharts, D3, data visualization patterns, real-time displays, metric grids, analytics components

## Tools
| Tool | Description |
|------|-------------|\n| `chart_generator` | Generate a chart component from data description and chart type |\n| `dashboard_layout` | Design a dashboard layout with metric cards, charts, and filters |\n| `data_transformer` | Transform raw data into chart-ready format with aggregations |\n| `realtime_display` | Create a real-time updating data display component |\n| `metric_card` | Generate stat card components with trend indicators and sparklines |

## Setup
```bash
npm install && npm run build
```
