# mama tax strategist MCP

Comprehensive tax strategy — entity comparison, holding companies, deductions, retirement optimization, audit risk, international tax

## Tools
| Tool | Description |
|------|-------------|\n| `entity_advisor` | Compare LLC vs S-Corp vs C-Corp tax implications for a business |\n| `holding_company_planner` | Design optimal holding company structure for multiple businesses |\n| `deduction_maximizer` | Generate comprehensive deduction checklist for a business type |\n| `estimated_tax_calculator` | Calculate federal and state quarterly estimated taxes |\n| `retirement_tax_optimizer` | Maximize retirement contributions across all tax-advantaged vehicles |\n| `international_tax_check` | Analyze treaty benefits, FBAR, and FATCA requirements |\n| `audit_risk_scorer` | Score audit risk 0-100 based on financial profile |\n| `year_end_checklist` | Generate month-by-month year-end tax optimization moves |

## Setup
```bash
npm install && npm run build
```
