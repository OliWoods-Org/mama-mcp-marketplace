# @mama/brand-strategist-mcp

**Brand Strategist MCP Server**

Build a brand that wins. Audit your brand health, generate complete style guides, brainstorm names, define your voice, and map your competitive positioning — all in minutes.

## Tools

| Tool | Description |
|------|-------------|
| `brand_audit` | Score brand health across 8 strategic dimensions with archetype identification |
| `style_guide_generator` | Generate complete style guide: colors, typography, logo rules, imagery, voice |
| `naming_brainstorm` | Brainstorm 8+ name candidates with scores, domain availability, and TM risk |
| `brand_voice` | Define brand voice with tone spectrum, do/don't rules, and before/after copy |
| `competitive_positioning` | Map positioning vs competitors with white space opportunities and statement |

## Usage

```json
{
  "mcpServers": {
    "brand-strategist": {
      "command": "npx",
      "args": ["-y", "@mama/brand-strategist-mcp"]
    }
  }
}
```

## Example Prompts

- "Audit the brand health of my 3-year-old SaaS company targeting B2B marketing teams"
- "Generate a style guide for a premium wellness brand with a bold, modern personality"
- "Brainstorm names for an AI-powered legal document tool targeting solo founders"
- "Define the brand voice for a fintech startup — audience is millennial investors, style: bold"
- "Map our competitive positioning vs Notion, Linear, and Asana in the productivity space"
