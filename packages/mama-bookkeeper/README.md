# mama bookkeeper MCP

AI bookkeeping — invoice scanning, expense categorization, P&L summaries, tax deductions, quarterly estimates

## Tools
| Tool | Description |
|------|-------------|\n| `scan_email_invoices` | Extract vendor, amount, date, and line items from email or receipt text |\n| `categorize_expense` | Auto-categorize a transaction with tax deductibility flag |\n| `monthly_summary` | Generate P&L summary from a list of transactions |\n| `tax_deduction_check` | Check if an expense is tax deductible for a given business type |\n| `quarterly_tax_estimate` | Calculate quarterly estimated tax payment from income and expenses |\n| `generate_invoice` | Generate a professional invoice from client and line item details |

## Setup
```bash
npm install && npm run build
```
